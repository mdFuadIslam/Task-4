## Key Features

- **Login:** Login with valid credentials.
- **Admin Panel:** Use system to block, delete and unblock users.

- **Backend:** Rails framework
- **Frontend:** Simple CSS.
- **Database:** MySQL
- **Deployment:** Deployed on Render.
